Aura — Login só e-mail + senha e CAPTCHA opcional (sem Google)
================================================================================

1) DESLIGAR GOOGLE (painel, não SQL)
   • Supabase → Authentication → Providers → Google → Disable

2) URLS (se um dia voltares a usar OAuth)
   • Authentication → URL Configuration → Redirect URLs
   • Não é possível configurar isto com SQL.

3) CAPTCHA (Turnstile recomendado, gratuito)
   a) Cria um site em https://dash.cloudflare.com → Turnstile → copia:
      - Site Key (pública) → cola em supabase-config.js como AURA_TURNSTILE_SITE_KEY
      - Secret Key (privada) → só no Supabase (passo b)
   b) Supabase → Authentication → Attack Protection
      • Ativa “Bot and abuse protection” / CAPTCHA conforme a UI atual
      • Escolhe Cloudflare Turnstile e cola o SECRET
   c) Se AURA_TURNSTILE_SITE_KEY estiver vazia no supabase-config.js, o widget
      não aparece e o login funciona sem CAPTCHA (útil em dev).

4) Ficheiro SQL
   • supabase/COLE_AUTH_EMAIL_CAPTCHA.sql — só texto explicativo + SELECT 1.
   • Não substitui cliques no painel.
