Profissionais parceiros · Cadastro

Este cadastro busca criar uma rede de apoio qualificada e empática. As informações serão analisadas pela equipa para garantir um atendimento alinhado ao acolhimento do Conta Mãe Atípica.

As mães terão disponível a agenda para consultas ou sessões com profissionais registados — valor fixo inicial R$ 49,90 (R$ 9,90 para o projeto · R$ 40,00 para o profissional).

Ao preencher este formulário, ficam cientes de que na plataforma o foco é o compromisso ético com a causa atípica: reduzir barreiras ao suporte, com sustentabilidade para o projeto e condição digna para o parceiro.
